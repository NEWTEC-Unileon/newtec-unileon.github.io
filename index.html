NEWTEC-Unileon Bioinformatic Pipelines

This website has been created to share all the bioinformatic pipelines employed in NEWTEC-Unileon research's group for different omics analysis. You can visit our institutional website to kwnow more about us and our work

We are trying to organize the web in a clear way to have the scripts for each pipeline in a github repository and the tutorial, ticks and further explanations in the current website.

Of course, you can share and modify the scripts if there are no commercial interest.
