Average Aminoacidic Identity

This pipeline uses EzAAI to calculate Average Aminoacidic Identity values between a set of genomes, preferably from the same genus. The first step is to install EzAAi following the instructions from https://github.com/endixk/ezaai:

conda install -c bioconda ezaai

Then, a list of the genomes to be compared must be done by:

ls GENOMES_FOLDER/ | sed “s/.fna//g” > list.txt

The .fna must be change according to the termination of your files, if they not finish by .fna. A very dirty and useful way to avoid this problem may be include all variations for fasta file terminations:

ls GENOMES_FOLDER/ | sed “s/.fna//g” | sed “s/.fasta//g” | sed “s/.fa//g” > list.txt

If you have installed ezaai in a conda environment (for example a conda environment named ezaai), you must activate it and run the following commands:

conda activate ezaai
mkdir faa_files
for x in $(cat list.txt); do ezaai extract -i GENOMES_FOLDER/${x}.* -o faa_files/${x}.faa; done
ezaai calculate -i faa_files/ -j faa_files/ -o AAI_output.txt -t NUMBER_THREADS
ezaai cluster -i AAI_output.txt -o AAI_cluster.nwk

The _AAIouput.txt contains AAI values on column named AAI, that can be plotted on… while the AAI_clsuter.nwk can be visualized on tree visualized software, or employed for further phylogenetic-tree plots by R.

Multi-Locus Sequence Typing (MLST)

Multi-Locus Sequence Typing (MLST) can be done for those species with MLST scheme, that you can check (here)[https://github.com/tseemann/mlst/blob/master/db/scheme_species_map.tab] or by using mlst --list within mlst conda environment (see below), using the pipeline in https://github.com/tseemann/mlst.

If you have installed mlst by conda, using conda install -c conda-forge -c bioconda mlst in mlst conda environment, you can run the pipeline by:

conda activate mlst
mlst –list > mlst_schemes.txt
for x in $(ls GENOMES_FOLDER); do mlst --scheme SELECTED_SCHEME GENOMES_FOLDER/${x} >> scheme_SELECTED_SCHEME.txt ; done

Core-genome and SNPs

The softwares Prokka and Panaroo

conda create -n panaroo python=3.9
conda activate panaroo
conda install -c conda-forge -c bioconda -c defaults 'panaroo>=1.3'
conda create -n prokka bioconda::prokka

ls GENOMES_FOLDER/ | sed “s/.fna//g” | sed “s/.fasta//g” | sed “s/.fa//” > list.txt
conda activate prokka
mkdir prokka
for x in $(cat list.txt); do prokka --outdir prokka/${x} --genus GENUS --cpus NUMBER_CPUS GENOMES_FOLDER/${x}.*; done
mkdir prokka_gff_files
for x in $(ls prokka); do cp prokka/${x}/*gff prokka_gff_files/${x}.gff; done
conda activate panaroo
mkdir results_panaroo
panaroo -i prokka_gff_files/* -o results_panaroo/ --clean-mode strict -t NUMBER_THREADS
panaroo-msa -o results_panaroo/ -a core --aligner mafft -t NUMBER_THREADS

The file _core_gene_alignmentfiltered.aln can be used for plotting the phylogenetic tree, based on core-genome alignment.

fasttree -nt -nj core_gene_alignment_filtered.aln > NJ.tree   #NJ method
fasttree -nt  core_gene_alignment_filtered.aln > core_gen.tree   #Maximum-likehood method
