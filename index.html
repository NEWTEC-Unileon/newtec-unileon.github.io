This website has been created to share all the bioinformatic pipelines employed in NEWTEC-Unileon research's group for different omics analysis.

We are trying to organize the web in a clear way to have the scripts for each pipeline in our github repository to facilitate their downloading; while tutorials, tips and further explanations are (or will be) located in the current website.

Of course, you can share and modify the scripts if there are no commercial interest.

If you have any question not hesitate to contact us: jcobd@unileon.es; efert@unileon.es
